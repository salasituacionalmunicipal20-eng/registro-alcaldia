Herramienta de Reparación e Inyección de Parches Demográficos

📋 DICCIONARIO LOGÍSTICO ABSOLUTO — CAPAS DE AUDITORÍA ACTIVAS:

1. 🛡️ Extracción Territorial Pura (1x1): El sistema lee estrictamente la Comuna, Comunidad, Calle y UBCH de la fila específica del habitante, validando sus datos antes de asociarlos a nadie. Si el campo viene vacío, se ignora para no borrar la información existente en Firebase.
2. 🏥 Motor Clínico Inteligente: Elimina diagnósticos falsos ("NO INDICA", "NINGUNA", "SANO") para mantener el inventario de salud impecable.
3. 🏠 Separación Semántica de Vivienda: Divide la arquitectura (Apto/Casa/Rancho) y Tenencia (Propia/Alquilada). Fuerza la condición a malo si detecta "rancho".
4. 👨‍👩‍👧‍👦 Vinculación Familiar Exacta: Tras asegurar que los datos propios del habitante están completos, el sistema verifica si es Jefe de Hogar. Si no lo es, lo ancla por debajo del último Jefe detectado y evalúa su género para inyectarle el rol exacto (esposa, hija, hijo, madre, etc.).
5. 🪪 Filtros de Identidad y Cédula: Extrae nacionalidad (E/V), elimina puntos y calcula matemáticamente si el habitante tiene edad suficiente para poseer cédula física.
6. ☁️ Operación Aislada Firebase: Este motor actúa exclusivamente sobre la base de datos principal, dejando la sincronización con Supabase a cargo manual del Administrador.
